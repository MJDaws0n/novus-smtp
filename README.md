# novus-smtp

Simple SMTP client.

A library for the [Novus](https://github.com/MJDaws0n/Novus) language, installable
via [Nox](https://github.com/MJDaws0n/Nox).

## Install

```sh
nox pull smtp
```

## Documentation

See [`docs.md`](docs.md) for the full API reference.

## Import

```novus
import lib/smtp smtp;
```
